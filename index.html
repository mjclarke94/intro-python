Introduction to Programming in Python

  1. 2019 James Grant (r.j.grant@bath.ac.uk), Research Software Engineer, University Bath

This repository contains the lesson ‘Introduction to Programming in Python’, derived from the Software Carpentry lessons Programming with Python and Plotting and Programming in Python, under CC BY 4.0 built for notebooks.azure.com. The lesson is delivered in the jupyter notebooks based build system developed by Jack Betteridge and James Grant for Advancing Research Computing at University of Bath.
